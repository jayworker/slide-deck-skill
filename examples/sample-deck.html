Color Rules

데이터 색 — 강조색과 분리

색 하나 = 의미 하나 — 강조와 데이터를 섞지 않는다
색 하나 = 의미 하나
강조 그린 ≠ 데이터 그린 — 같은 슬라이드에서 혼용 금지
색 + 형태 이중 인코딩
점선/실선 · ✓/✗ · 숫자 병기 — 색만으로 싣지 않는다
레드는 끝까지 금지 전용
음수·시리즈 4에도 쓰지 않는다
